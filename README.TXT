Trabajo Práctico N°2 - Ejercicio 1
Videojuego Atoms
Autores: Martinez Gustavo - Tissera Adrian

REQUISITOS:
1-	Aunque el programa fué escrito, compilado y testeado en Ubuntu 14.04, no debiera presentar 
		ningun inconveniente ejecutarlo en otro sistema operativo.

2- 	El programa fue escrito en Java 8.0 por lo que se recomienda ejecutarlo en esa versión
		(No fue testeado en versiones de Java anteriores)

EJECUCIÓN:
Para ejecutar la aplicación desde la linea de comandos debe encontrarse dentro de la carpeta 
'dist' la cual se encuentra dentro de la carpeta del proyecto y ejecutar el siguiente comando:

java -jar "Atoms.jar" 

Nota:
Para distribuir este proyecto, comprima la carpeta dist (incluyendo la carpeta lib)
y distribuya el archivo comprimido.
